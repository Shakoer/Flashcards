# Flashcards
A flashcard website.

- Flash cards stored in JSON
- make own flashcards:
  ```
  {
    "subject" : [
      "Q": "question?"
      "A": "answer"
    ]
  }
  ```
- progress tracking.
- press to reveal and press to go to next card.
